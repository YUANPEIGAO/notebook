☁️ GitHub 同步设置

第一步：生成 Personal Access Token

  1. 登录 GitHub，点击右上角头像 → Settings
  2. 左侧菜单选择 Developer settings
  3. 点击 Personal access tokens → Generate new token (classic)
  4. 勾选权限：repo (Full control of private repositories)
  5. 点击 Generate token，复制生成的 Token

第二步：配置笔记应用

  1. 点击侧边栏的"⚙️ 设置"按钮
  2. 粘贴刚才生成的 Token
  3. 输入你的 GitHub 用户名
  4. 输入你想存储笔记的仓库名称（如不存在会自动创建）
  5. 点击"保存并测试"

第三步：开始同步

配置成功后，点击"同步到 GitHub"按钮即可将本地笔记上传到仓库
