常用块级容器标签完整总结（文本格式，含作用 + 用法）
所有标签都是块级元素：独占一行、宽度默认铺满父级、可设置宽高 / 内外边距，专门用于页面布局、分区、包裹内容。
一、语义化块级标签（推荐优先使用，比 div 更有意义）
header作用：定义页面 / 区域头部（导航、logo、标题区）实例：<header>网页头部</header>
nav作用：定义导航栏（菜单、跳转链接组）实例：<nav>主导航</nav>
main作用：定义页面主体内容（一个页面只用一次）实例：<main>核心内容区</main>
section作用：定义页面独立区域 / 章节（文章段落、板块）实例：<section>商品介绍区</section>
article作用：定义独立内容块（文章、帖子、卡片）实例：<article>一篇博客</article>
aside作用：定义侧边栏 / 附属内容（广告、相关推荐）实例：<aside>右侧边栏</aside>
footer作用：定义页面 / 区域底部（版权、联系方式）实例：<footer>页面底部</footer>
address作用：定义联系地址 / 作者信息实例：<address>北京市朝阳区</address>
二、通用内容块级标签（传统布局常用）
p作用：定义段落（自带上下外边距）实例：<p>这是一段文字</p>
ul作用：定义无序列表（配合 li 使用）实例：<ul><li>列表1</li></ul>
ol作用：定义有序列表（带数字排序）实例：<ol><li>第一项</li></ol>
li作用：定义列表项（必须放在 ul/ol 内）实例：<li>列表内容</li>
dl / dt / dd作用：定义自定义列表（术语 + 说明）实例：<dl><dt>名称</dt><dd>解释</dd></dl>
h1 ~ h6作用：定义标题（h1 最大，h6 最小）实例：<h1>主标题</h1>
form作用：定义表单区域实例：<form>输入框、按钮</form>
table作用：定义表格实例：<table><tr><td>内容</td></tr></table>
三、块级容器标签使用规则
语义优先：能用 header /nav/main /footer/section /article，就不用 div
div 是万能兜底：没有合适语义标签时，用 div
所有块级标签可互相嵌套（标题、段落尽量不嵌套大块容器）
都可加 class / id 用于 CSS 样式和 JS 操作
四、最简记忆表
头部：header
导航：nav
主体：main
区域：section
文章：article
侧边：aside
底部：footer
段落：p
列表：ul /ol
标题：h1~h6
万能容器：div
总结
块级标签 = 独占一行 + 可设宽高 + 用于布局
语义标签更利于 SEO、可读性、维护
div 是无语义通用容器，其他标签有专门用途
日常开发：语义标签 + div 组合使用
<hr> 是 HTML 中水平线分隔符标签，属于单标签、块级元素，核心作用是在页面中插入一条水平线，用于分隔不同内容区块，让页面结构更清晰，无默认语义，是布局中常用的分隔工具。
class="divider" 是 HTML 元素的类名属性，divider 英文意为分隔线 / 分隔器，是前端开发中约定俗成的语义化类名，专门用于标识页面分隔元素（水平线、间隔块、装饰分隔条）。
<hr class="divider">
、最常用搭配标签
配合 <hr>（最推荐）
写法：<hr class="divider">
配合 <div>（通用分隔块）
写法：<div class="divider"></div>
